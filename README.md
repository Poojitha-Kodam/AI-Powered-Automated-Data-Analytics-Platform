# AI-Powered Automated Data Analytics Platform

An AI-powered analytics platform built using FastAPI, React, SQLite, and Google Gemini.

## Features

- Dataset Upload
- AI Chat Assistant
- SQL Query Generator
- Data Cleaning
- Data Profiling
- Dashboard Builder
- Forecasting
- Machine Learning Playground

## Tech Stack

Frontend:
- React
- Vite
- TailwindCSS

Backend:
- FastAPI
- SQLAlchemy
- SQLite
- Google Gemini API

## Run

Backend

```bash
uvicorn main:app --reload
```

Frontend

```bash
npm run dev
```
